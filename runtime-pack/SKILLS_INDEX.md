# E2B Agent Runtime Skills Index

Available skills for the E2B Worker Sandbox:

- `tool-routing`: Guides tool routing between PTY, one-shot commands, and repository file tools.
- `security-and-trust`: Security policy enforcement, secret scanning, credential boundary protection.
- `repository-discovery`: Stack detection, manifest analysis, governance discovery.
- `task-modes`: Feature, bugfix, issue, and audit execution modes.
- `change-execution`: Safe code changes, diff review, atomic commits.
- `validation-and-ci`: Testing, linting, type-checking, build verification.
- `architecture-analysis`: System architecture mapping, module boundaries.
- `context-checkpoints`: Task state persistence across steps.
- `pr-creation`: Branch publication preflight, pull request handoff formatting.
