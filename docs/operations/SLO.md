# Service Level Objectives (SLOs)

This document establishes the staging target metrics for the Remote MCP Controller.

| Metric | Target | Time Window |
|---|---|---|
| **Controller Availability** | >= 99.5% | 30 Days |
| **MCP Request Success Rate** | >= 99% | 30 Days |
| **MCP p95 Latency** | <= 500ms | 30 Days |
| **Worker Creation Success Rate**| >= 95% | 30 Days |
| **Worker Creation p95 Duration**| <= 15s | 30 Days |
